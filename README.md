### CVRF Engine (Full-Stack Belief System)

All custom, zero external ML dependencies â€” no scikit-learn, no TensorFlow, no framework lock-in:

| Layer | Implementation |
|-------|---------------|
| **Concept Extraction** | `ConceptExtractor.ts` â€” extract investment beliefs from factor analysis |
| **Belief Management** | `CVRFManager.ts` â€” orchestrate belief lifecycle, conviction tracking |
| **Reinforcement** | `BeliefUpdater.ts` â€” reinforce correct beliefs, weaken incorrect ones |
| **Episode Tracking** | `EpisodeManager.ts` â€” temporal windows, regime change detection |
| **Persistence** | `PersistentCVRFManager.ts` â€” Supabase-backed survival across restarts |
| **Integration** | `integration.ts` â€” connect beliefs to optimizer and backtest runner |

The system automatically learns, reinforces, and manages beliefs via episodes.

```env
# CVRF Configuration
CVRF_LEARNING_RATE=0.1
CVRF_DECAY_RATE=0.05
CVRF_MIN_CONVICTION=0.1
CVRF_MAX_CONVICTION=1.0
CVRF_EPISODE_WINDOW=30
```
